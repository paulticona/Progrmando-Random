Pseudoclases

Estilos especiales dependiendo del contexto, posicion o estado.

:root

La pseudoclase de CSS :root selecciona el elemento rais de un documento aplicado a HTML. :Root representa el elemento html y es identico al selector html excepto que su nivel de especificidad es mayor y sive para declarar variables css.

:hover

La pseudoclase de CSS :hover se presenta cuando el usuario coloca el puntero sobre algun elemento, pero no necesariamente esta activo. en pantalla de moviles touch :hover es problematico.

:active

La pseudoclase de CSS :active cubre en el caso de que el elemento esta siendo activado por el usuario. Cuando se interactua con el raton, se corresponde en el momento que el usuria presiona el boton del raton hasta el momento en que lo suelta a menudo es usando en los elemntos A Y Button tambien el orden importa:
1. :link{...}
2. :visited{...}
3. :hover{...}
4. :active{...}

:target

La pseudoclase de CSS :target representa el elemento unico, si existe alguno con ID coincidente con elm identificador de fragmentos de la URL del documento.

  • :last-of-typeselleciona el elemento indicado que sea elultimo hijo sin inportar la jerraquia dentro del elemneto padre.
